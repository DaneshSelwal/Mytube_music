# MyTubeMusic 🎵

A stunning, premium, modern Android music player featuring a gorgeous Glassmorphism design, spinning vinyl animations, and real-time auto-scrolling lyrics. Built with Kotlin and Jetpack Compose.

## ✨ Features

*   **Modern UI/UX**: Immersive gradients, beautiful glassmorphism aesthetic, and perfectly smooth Shared Element transitions.
*   **Spinning CD Animation**: A mesmerizing spinning CD with custom iridescent gradients that reacts to playback state (play/pause).
*   **Library Management**: Intelligent local media scanning that automatically parses track and artist data directly from your device.
*   **Synchronized Lyrics**: Built-in `.lrc` parser that automatically syncs and smoothly auto-scrolls lyrics alongside your playing music!
*   **Advanced Playback Controls**:
    *   Sleep Timer
    *   Adjustable Playback Speed
    *   Shuffle & Repeat modes
    *   Seamless Timeline Scrubbing
*   **Architecture**: Built using the latest Android best practices—MVVM, StateFlows, Room Database, and Media3 for robust background playback.

## 🛠️ Tech Stack

*   **Language**: Kotlin
*   **UI Toolkit**: Jetpack Compose
*   **Playback Engine**: AndroidX Media3 (ExoPlayer)
*   **Architecture**: MVVM (Model-View-ViewModel)
*   **Database**: Room
*   **Coroutines & Flow**: For asynchronous operations and state management

## 🚀 Getting Started

### Prerequisites
*   Android Studio (Latest version recommended)
*   Android SDK 34
*   JDK 17

### Installation
1.  Clone the repository:
    ```bash
    git clone https://github.com/DaneshSelwal/Mytube_music.git
    ```
2.  Open the project in Android Studio.
3.  Sync the project with Gradle files.
4.  Build and run the application on an emulator or physical device.
